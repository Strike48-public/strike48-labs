Strike48 is a native desktop platform that brings your developer tools together in a single window. Built with Rust and Dioxus, it provides a connector architecture where each tool runs as an isolated process, communicating with the host shell over IPC.

## Products

### StrikeHub

The unified desktop shell. StrikeHub discovers, launches, and manages connector applications — rendering each as an embeddable panel inside a single native window. It handles authentication, WebSocket bridging, and health monitoring so connectors can focus on their domain logic.

[Get started with StrikeHub →](/strikehub/)

### KubeStudio

Kubernetes cluster browser and management tool. Browse namespaces, inspect workloads, view pod logs, and manage deployments from a native desktop interface. KubeStudio runs as a StrikeHub connector or as a standalone application.

[Get started with KubeStudio →](/kubestudio/)

### Pick

Connector marketplace and registry for the Strike48 ecosystem. Browse available connectors, install new tools, and manage your connector catalog.

[Get started with Pick →](/pick/)

## Architecture at a Glance

Strike48 uses a **host + connector** model:

- **StrikeHub** is the host shell — a Dioxus desktop app with a sidebar, content area, and status bar.
- **Connectors** (KubeStudio, JiraStudio, GitLabStudio, etc.) are independent processes that serve their UI via HTTP and communicate with the host over Unix domain sockets.
- **Pick** provides a marketplace for discovering and managing connectors.

Each connector runs in its own process for isolation — a crash in one tool doesn't affect the shell or other connectors. Communication happens over Unix domain sockets, so no TCP ports are exposed to the network.

## Tech Stack

| Component | Technology |
|-----------|------------|
| Language | Rust (2024 edition) |
| UI Framework | Dioxus 0.6 |
| WebView | Wry |
| HTTP Server | Axum 0.7 |
| Async Runtime | Tokio |
| IPC Transport | Unix domain sockets |
| Authentication | OIDC / OAuth (Keycloak) |
| Configuration | TOML |
