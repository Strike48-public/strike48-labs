Pick is a lightweight, headless penetration testing agent built with Dioxus. It runs as a StrikeHub connector or standalone binary, providing network reconnaissance, device discovery, and remote tool execution capabilities.

## What It Does

- **Network scanning** — TCP port scanning with concurrent connections
- **Device discovery** — ARP table reading, mDNS service discovery, UPnP/SSDP enumeration
- **Traffic capture** — Network packet capture (requires elevated privileges)
- **WiFi scanning** — WiFi network enumeration
- **System recon** — Device and system information gathering
- **Screen capture** — Screenshot capture as base64 PNG
- **Command execution** — Remote shell command execution
- **Bi-directional control** — Tools triggered manually via UI or remotely via Prospector Studio API

## Architecture

Pick is designed as a droppable payload — a single binary that can be deployed on any system without requiring a GUI, windowing system, or desktop environment.

It operates in two modes:
- **Standalone mode** — Connects directly to Prospector Studio over gRPC
- **StrikeHub IPC mode** — Runs as a managed connector within StrikeHub, communicating over Unix domain sockets

When running inside StrikeHub, Pick serves a full workspace UI (dashboard, tools, files, shell, logs, chat) via Dioxus LiveView.
